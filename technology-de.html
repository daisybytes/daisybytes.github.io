Technologie

DAISYBYTES setzt primär auf die folgenden Kerntechnologien für die Umsetzung Ihres Projektes:

Clojure

Clojure ist eine allgemeine, dynamische, funktionale Programmiersprache, die in der Java Virtual Machine läuft und ist eng in die Java Runtime integriert ist. Clojure Software kann daher schnell und einfach auf jeder Platform eingesetzt werden, die über native Java Unterstützung verfügt. Dank Clojures bereits im Grunddesign verankerten Einfachheit, sowie exzellenter Spezifizierungs- und Testfunktionalitäten ist ein erster Prototyp bereits nach kurzer Zeit verfügbar und kann dann sukzessiv anhand von Kunden- und Anwenderfeedback weiter in Form gebracht werden. Mit diesen Eigenschaften ist Clojure die perfekte Technologie um mit agilen Entwicklungstechniken kombiniert zu werden, welche in für Sie speziell angepasster Form in allen DAISYBYTES Projekten zum Einsatz kommen sollen. Mit Clojure als erste Kerntechnologie strebt DAISYBYTES eine saubere, korrekte, leistungsfähige aber dennoch einfache Lösung Ihrer Anforderungen an.

ClojureScript

ClojureScript ist ein Compiler für Clojure der JavaScript erzeugt. ClojureScript kombiniert die Reichweite der JavaScript platform mit der Flexibilität und Interaktivität von Clojure sowie der Gesamtoptimierung von Google Clojure und bildet somit die mächtigste Sprache zur Programmierung von Webseiten. Es existieren Implementierungen für häufig verwendete Bibliotheken wie z.B. React oder Om, die nahtlos in Ihre Projektlösung integriert werden können. Besonders Google fördert die Entwicklung von ClojureScript als führende Kraft und setzt ClojureScript bereits erfolgreich in seiner Online-Suite ein. ClojureScript ist die zweite von DAISYBYTES Kerntechnologien, da ClojureScript als Teil des Clojure Gesamtpaketes Synergieeffekte erzeugt, die Entwicklungszeit reduzieren und die Qualität des Endresultats erhöhen.

Datomic

Datomic (eine Implementierung von Datalog) ist eine transaktional, verteilte, kommerzielle Datenbank entwickelt von der Firma Cognitect. Sie bietet flexible Datenmodelle, elastische Skalierung und ausdrucksstarke Datenbankanfragen. Datomic wurde von Grund auf für den Einsatz innerhalb der Amazon Cloud entwickelt, kann jedoch auch falls gewünscht direkt auf Ihrer internen Infrastruktur aufgesetzt werden. Unterstützung für Verschlüsselung, VPC Isolierung, Logging wie auch Verteilung via CloudFormation sowie komplett-integrierte Managementfunktionen innerhalb der AWS Konsole gehören zum Standard bei Datomic. Innerhalb einer Datomic Datenbank werden sämtliche Daten im Zeitkontext gespeichert, nichts kann verloren gehen. Dadurch können alle Änderungen genau nachvollzogen werden, was insbesonders bei der Auswertung von finanziellen, medizinischen oder allgemeinen Bestandsdaten Vorteile bietet. Datomic reduziert unnötige Komplexität innerhalb des Systems und damit auch die Entwicklungszeit.

Free Open-Source Software

DAISYBYTES setzt primär auf den Einsatz von freier (wie in Freiheit) quelltext-offener Software (FOSS) um Ihnen innerhalb Ihres Projektes folgende Vorteile zu ermöglichen:

  • Bessere Sicherheit
    Laut "Linus' Gesetz" von Linux Begründer Linus Torvalds können Defekte innerhalb einer Software umso schneller behoben werden, je mehr Entwickler Einsicht in der Code haben und selbigen testen können. Im Gegensatz zu den reinen Behauptungen von Anbietern proprietärer Software erlaubt lediglich die komplette Einsicht in den gesamten Code einer Software volles Vertrauen in die Art und Weise, welche Ihrer Daten, wie genau, zu welchem Zweck verwendet werden.
  • Höhere Qualität
    Unzählige passionierte Entwickler und Nutzer streben nach kontinuierlicher Weiterentwicklung von FOSS Projekten aller Art und tragen tägliche Innovationen bei. FOSS Projekte sind den Zielen ihrer Nutzer desweiteren so nah wie nur möglich, da diese Nutzer selbst aktiv an der Gestaltung der Software teilnehmen können.
  • Mehr Freiheit
    Nutzer von FOSS haben komplette Kontrolle und können selbst entscheiden, welche Version sie verwenden wollen, ob und wann genau sie diese nachrüsten möchten. Desweiteren können sie die Software frei nach ihren eigenen Wünschen und Anforderungen anpassen. Aktive Gemeinschaften von Entwicklern und Nutzern auf der ganzen Welt bieten zu jeder Zeit kostenlose Unterstützung bei Problemen aller Art und helfen Ihnen durch technische Dokumentation bis hin zu Live-Chats. FOSS verwendet und hält sich an offene Standards und Formate, weshalb sich die Integration in ein System von multiplen, unterschiedlichen Komponenten einfacher gestaltet und somit wertvolle Projektzeit sparen kann.
  • Keine Herstellerbindung
    Aufgrund der Natur des FOSS Entwicklungsmodells, mit einem Fokus auf offenen Standards, ist eine Herstellerbindung wesentlich geringer ausgepräckt als bei der Verwendung von proprietärer Software.
  • Geringere Kosten
    Lizens- oder Supportkosten, sowie Gebühren für Aktualisierungen oder Folgeversionen wie auch indirekte Kosten durch die Fixierung auf einen einzelnen Anbieter existieren nicht bei der Verwendung von FOSS. Erste Evaluierungen können komplett ungebunden und selbstverständlich kostenlos durchgeführt werden, bis Sie sich genau darüber im Klaren sind, welcher Weg für Sie der beste ist.
